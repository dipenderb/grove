# Logs

My own daily activity log — what **I** did operating this OS, not the user's journal. One file per day: `YYYY-MM-DD.md`, append-only, frontmatter-first, plain bullets with links. **When:** every turn I act (active trigger, not "session end"). Protocol: `../../_system/skills/agent-log.md`.
