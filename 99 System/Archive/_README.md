# 99 System/Archive

The canonical home for inactive items — finished/dropped projects, dormant material, past journal years. **Organised object-wise** (by type, plural, no folder numbers): `Archive/Projects/`, `Archive/Resources/`, etc. The Journal archives **by year**: `Archive/Journal/YYYY/YYYY-MM mmm/`.

Rule: **archive, never delete.** Wikilinks to archived objects still resolve, so history and backlinks survive. See `99 System/_system/skills/archive.md`.

> Describes purpose, not contents — read the folder for what's actually archived.
