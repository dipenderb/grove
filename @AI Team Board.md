# AI Team Board

The shared work board for you and your AI team — **you both edit this.** Add or reprioritise requests; the agent keeps the statuses current and owns execution order. (The agent's private reasoning, logs, and memory stay in `99 System/_agent/`.)

**How work flows:** drop a file/request in `01 AI Team Inbox/` *or* add a line under **Queued** below → the agent routes and works it → file outputs land in `02 AI Team Deliverables/` for your review, and the agent summarises in chat.

> Use plain `-` bullets here (not `- [ ]`) so team work doesn't mix into your personal task lists. Move a line between sections as its status changes.
>
> 💡 **Kanban view (optional):** the `## ` sections are lanes and each `-` line is a card. Install the **Kanban** community plugin, then right-click this note → *Open as Kanban board* to drag cards between lanes. It stays plain Markdown either way.
