# AI Team Deliverables

The **single place** your AI team drops file outputs for you to review — drafts, research write-ups, summaries, generated files. Check here for finished work; the agent also summarises each delivery in chat with a link.

**Review surface only.** Files stay here for you to read — **nothing is auto-filed into your vault from here**. If you want a deliverable kept as part of your system, just say so and the agent files it as a proper object (a `note`/`resource`, linked by topic). Clear the rest whenever.

Distinct from the agent's private workspace in `99 System/_agent/`.
