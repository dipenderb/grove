# 👋 START HERE

This is **Grove** — one folder that runs your whole life and work, with an assistant that builds it around *you*. You don't need to be technical. Pick one of the two ways below, then just say **hi**.

**You'll need:** an AI assistant you already use — Claude Code, Gemini CLI, or Codex.

---

## Option A — Obsidian  *(recommended — nicer app, visual dashboards)*

1. Download **Obsidian** (free): https://obsidian.md
2. Open Obsidian → **"Open folder as vault"** → choose this folder.
3. **Turn on community plugins:** Settings → **Community plugins** → click **"Turn on community plugins"** (this exits Restricted Mode — required before you can install anything).
4. **Install the Terminal plugin:** still in Community plugins → **Browse** → search **"Terminal"** → **Install** → **Enable**.
   *(Optional but great: also install **Dataview** and **Tasks** the same way.)*
5. **Configure Terminal** (Settings → **Terminal**):
   - set **Renderer** to **DOM** *(if you later see junk/garbled characters, this is the setting to change)*,
   - set the **default profile** to **Integrated**.
6. Open the **Terminal** panel (Command palette → *"Terminal: Open terminal"*, or the ribbon icon).
7. In the terminal, **type the command that starts your AI** — e.g. **`claude`**, **`gemini`**, or **`codex`** — and press **Enter**.
8. Once it's running, just type **hi**.

## Option B — VS Code or any editor  *(if you live in VS Code or a terminal)*

1. Open this folder in **VS Code** (or open a terminal in this folder).
2. In a terminal **in this folder**, type the command that starts your AI — e.g. **`claude`**, **`gemini`**, or **`codex`** — and press **Enter**.
3. Once it's running, just type **hi**.

## Then what?

The assistant takes over — it greets you, asks a few simple questions, and **builds your system around your life**. You mostly just talk to it.

- You **can't break anything.**
- New to all this? Say so — it explains everything in plain language, step by step.
- No command to remember: **just say hi** and it begins.

*(If the terminal text ever looks garbled, just tell the assistant — it'll fix it for your computer.)*
